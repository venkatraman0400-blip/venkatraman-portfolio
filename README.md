## 🚀 Local Development

```bash
# 1. Clone the repo
git clone https://github.com/venkatraman0400-blip/venkatraman-portfolio.git
cd venkatraman-portfolio

# 2. Install dependencies
npm install

# 3. Start dev server
npm run dev

# 4. Build for production
npm run build
```
